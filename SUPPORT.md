# Support

## Getting Help

- **Bug reports & feature requests:** Open an issue using the appropriate
  template in this repository.
- **Questions & discussion:** Use this repository's **Discussions** tab for
  general questions, setup help, and ideas.

## Security Issues

**Do not open a public issue for security vulnerabilities.** Use
**Security → Report a vulnerability** in this repository instead. See
[SECURITY.md](SECURITY.md) for details.

## Out of Scope

The following are **not** supported through this project:

- **GitHub account issues** — contact [GitHub Support](https://support.github.com/).
- **GitHub Copilot billing or licensing** — see
  [GitHub Copilot documentation](https://docs.github.com/en/copilot).
- **Docker installation or host OS configuration** — refer to
  [Docker documentation](https://docs.docker.com/).
- **Private deployment troubleshooting** — Hangar is provided as-is. We
  cannot debug your specific host, network, or credential configuration.

## Response Times

This is a volunteer-maintained open-source project. There are no guaranteed
response times. We appreciate your patience.
